// C++: High-performance GEMM with TensorCraft
#include "tensorcraft/kernels/gemm.hpp"
#include "tensorcraft/memory/tensor.hpp"

// Create tensors (RAII-managed GPU memory)
tensorcraft::FloatTensor A({4096, 4096});
tensorcraft::FloatTensor B({4096, 4096});
tensorcraft::FloatTensor C({4096, 4096});

// Optimized GEMM: C = α(A × B) + βC
tensorcraft::kernels::gemm(
    A.data(), B.data(), C.data(),
    4096, 4096, 4096);
# Python: Simple NumPy-compatible API
import tensorcraft_ops as tc
import numpy as np

# Matrix multiplication
A = np.random.randn(4096, 4096).astype(np.float32)
B = np.random.randn(4096, 4096).astype(np.float32)

# FlashAttention-style operation
Q = np.random.randn(32, 128, 64).astype(np.float32)
K = np.random.randn(32, 128, 64).astype(np.float32)
V = np.random.randn(32, 128, 64).astype(np.float32)

output = tc.flash_attention(Q, K, V)

What the repository covers

Representative kernels, readable structure, and a smaller maintenance surface

GEMM

Matrix multiplication with progressive optimization from naive → tiled → double buffer → Tensor Core (WMMA).

85-95% cuBLAS

Attention

FlashAttention with memory-efficient computation, RoPE positional encoding, and MoE Router implementations.

80-90% cuDNN -80% Memory

Normalization

LayerNorm, RMSNorm, BatchNorm, and Softmax with optimized fusion strategies for modern GPUs.

90-95% cuDNN

Convolution

2D/3D convolutions with Im2Col, Winograd, and depthwise separable optimizations.

75-85% cuDNN

Sparse Operations

CSR/CSC format support, SpMV, SpMM, and structured sparsity patterns optimized for sparse matrices.

High Sparsity

Quantization

INT8 and FP8 (CUDA 12.0+) quantized operations for reduced precision with minimal accuracy loss.

2x Speedup CUDA 12+
