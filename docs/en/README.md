# TensorCraft-HPC Documentation Hub

Welcome to the **TensorCraft-HPC** documentation hub. This is a modern C++/CUDA AI high-performance computing kernel library for learning, validating, and implementing core algorithms including GEMM, Attention, Convolution, Normalization, Sparse Operators, and Quantization.

## Documentation Structure

```
docs/
├── getting-started/    # Quick start and installation guides
├── guides/             # Architecture design and optimization guides
├── api/                # API reference documentation
├── examples/           # Code examples and tutorials
└── reference/          # Contributing guidelines, changelog, community standards
```
