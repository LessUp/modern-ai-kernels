# 学院

学院部分的目标不是只教你怎么编译，而是教你如何把这个仓库当作一个系统来阅读。

## 你会在这里学到什么

- 仓库如何组织 kernels、memory primitive 与硬件能力检测
- 优化路径如何从朴素版本演进到 Tensor Core aware 实现
- 应该怎样结合正确的方法论去评估 benchmark 结论

1. [快速开始](/zh/guides/quick-start)
2. [架构解读](/zh/guides/architecture)
3. [现代 C++ 与 CUDA](/zh/guides/modern-cpp-cuda)
4. [基准测试](/zh/guides/benchmarking)
5. [示例](/zh/examples/)
